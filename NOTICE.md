# Academic and authorship notice

The historical laboratory work described in this repository was completed as a four-person academic team in the Automatic Control I laboratory at Friedrich-Alexander-Universität Erlangen-Nürnberg.

The public code in this repository is a clean-room portfolio reconstruction based on the verified engineering topics and equations. It is not a dump of the original team submission. University instruction sheets, raw reports, student identifiers, unofficial messages, duplicate archives, and files with unclear individual authorship are intentionally excluded.

The repository must not be used to claim sole authorship of every calculation, model, script, or report section from the historical laboratory work. A project-specific contribution record should be maintained when individual responsibilities are confirmed.
